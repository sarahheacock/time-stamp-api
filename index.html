FreeCodeCamp API Basejump: Timestamp Microservice

User stories:
    I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: Jan 1, 2016)
    If it does, it returns both the Unix timestamp and the natural language form of that date.
    If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

https://r282-timestamp-api.herokuapp.com/Dec%2015,%202015
https://r282-timestamp-api.herokuapp.com/1450137600

Example output:

{ "unix": 1450137600, "date": "Dec 15, 2015" }
